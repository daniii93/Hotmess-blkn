## Go-Live Flow

1. User registrieren.
2. Profilbild hochladen.
3. Verifikation starten.
4. Admin verifiziert.
5. Event veroeffentlichen.
6. Ticket kaufen.
7. Add-on buchen.
8. E-Mail erhalten.
9. QR in App oeffnen.
10. Scanner scannt Ticket.
11. Doppel-Scan wird blockiert.
12. Admin sieht Umsatz.
13. Admin exportiert Gaesteliste.
