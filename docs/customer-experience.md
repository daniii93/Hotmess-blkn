# HotMess Customer Experience Map

1. Discover:
   Goal: understand HotMess as premium community experience.
   Touchpoints: homepage, events, gallery, social sharing.

2. Ticket:
   Goal: secure access.
   Touchpoints: event detail, tickets, app wallet.

3. Hotel:
   Goal: remove travel friction.
   Touchpoints: hotels, packages, concierge.

4. Membership:
   Goal: unlock preference.
   Touchpoints: Passport, account, app, benefits.

5. Event:
   Goal: arrive informed and respected.
   Touchpoints: app guide, safety, host notes.

6. Community:
   Goal: return beyond a single event.
   Touchpoints: circle, brunch, ambassadors.

7. Repeat guest:
   Goal: increase loyalty and personalization.
   Touchpoints: rewards, referrals, CRM.

8. Ambassador:
   Goal: become a local brand signal.
   Touchpoints: ambassador program, referrals, operator system.
