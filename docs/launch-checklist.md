# HotMess Launch Checklist

- Domain configured.
- SSL active.
- Privacy, terms, imprint and cookies reviewed by legal counsel.
- Stripe connected and tested.
- Supabase or production database connected.
- E-mail provider connected.
- Analytics configured.
- Error tracking configured.
- Backup and restore tested.
- Security review completed.
- Mobile test completed.
- Performance test completed.
- Admin password rotated.
- First event content approved.
- Partner and hotel benefits verified.
