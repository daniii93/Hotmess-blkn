# HotMess Backup Strategy

Database backup:

- Daily automated database backup.
- Weekly manual restore test.
- Store encrypted backup outside primary hosting account.

Media backup:

- Back up uploaded images, profile photos, gallery media and documents.
- Keep at least 30 days of versions.

Recovery:

- Identify incident owner.
- Restore database.
- Restore media.
- Validate login, admin, forms and public routes.

Responsibilities:

- Founder: final go/no-go.
- Platform admin: recovery execution.
- Legal/ops: user communication when required.
