# HotMess Design System

Colors:

- Primary background: #080706.
- Surface: #14110f.
- Text: #f6efe5.
- Muted: #b7a99b.
- Accent gold: #e5c38c.
- Wine accent: #741f28.

Typography:

- Large editorial heroes.
- Tight, clear operational headings.
- No negative letter spacing.

Spacing:

- Generous page spacing.
- Compact but breathable admin cards.

Radius:

- Cards and panels: 8px or less.

Components:

- Hero: clear claim, short supporting copy, strong CTA.
- Cards: image or KPI first, clear status, one action.
- Badges: quiet, uppercase, gold or muted.
- Forms: large touch targets, clear labels, calm focus state.
- Tables: readable, dense, premium status badges.
