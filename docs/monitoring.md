# HotMess Monitoring

Prepared monitoring stack:

- Error tracking: Sentry or equivalent.
- Uptime monitoring: public pages, login, admin, forms, API health.
- Logging: security events, form submits, admin changes, revenue events.
- Analytics: privacy-aware page and conversion tracking.

Launch requirement:

- Configure production provider.
- Define escalation owner.
- Review error logs daily during launch week.
